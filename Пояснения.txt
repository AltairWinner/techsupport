Роли:
0 - не активированный аккаунт
1 - аккаунт администратора. Администратор может активировать и деактивировать пользователей.
2 - аккаунт сотрудника техподдержки. Имеет доступ к тикетам.


====================

ДЛЯ РАБОТЫ НЕОБХОДИМО:
1) Установить систему на веб-сервер (например, для XAMPP надо закинуть в папку htdocs)
2) Запустить Apache и MySQL
3) Создать в MySQL (например, с помощью PHPMyAdmin) пользователя с учётными данными:
1 способ
Имя пользователя: techsupportsystem
Пароль: techsupport

Учетные данные меняются в файле php/dbconnection.php

2 способ
Можно использовать SQL код:
GRANT SELECT, INSERT, UPDATE, DELETE, FILE ON *.* TO `techsupportsystem`@`localhost` IDENTIFIED BY PASSWORD '*08C069C4402412E866F24A42192D63E874169F2D';
Он автоматически создаст аккаунт, используемый системой.

Для работы системы необходима база данных. Скрипт для её создания называется techsuppodtdb.sql. Нужно импортировать его с помощью PHPMyAdmin.




После импорта в системе будет изначально создан аккаунт администратора 
Логин: admin 
Пароль: admin

Рекомендуется сменить данные данного пользователя в случае использования системы для Вашего бизнеса.
(Шутка, никто её не собирается использовать по назначению, только сдать преподавателю и всё)
